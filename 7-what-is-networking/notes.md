## Key Commands
- `ping 8.8.8.8` - Sends ICMP echo requests to Google DNS.
- `ping <IP>` - Tests connectivity to a specific IP address.

## Protocol Used
- Ping uses **ICMP (Internet Control Message Protocol)**
