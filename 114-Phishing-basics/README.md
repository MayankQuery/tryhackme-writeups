# Phishing Basics

This repository documents my completion of the TryHackMe room **Phishing Basics**, which explores phishing as a social engineering technique used to gain initial access during penetration testing.

---

- Platform: TryHackMe
- Room Name: Phishing Basics
- Duration: ~45 minutes

- Understand the structure of a phishing campaign
- Explore tools used in phishing engagements
